# billing-software

A modern, IRD‑compliant billing and invoicing system built for Nepalese businesses. This software is designed to handle VAT billing, invoice numbering, customer management, and reporting with reliability, auditability, and ease of use.

Tech Stack

Backend

Node.js / TypeScript
ExpressJS
PostgreSQL (VAT‑safe relational storage)

Frontend

React (with modern hooks)
Tailwind CSS for clean UI

DevOps & Tooling

Docker & Docker Compose
Git‑based migrations
Environment‑based configuration
